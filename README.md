# Love Letter
It's the board game

## Frontend

### Install dependencies
```
cd ui
npm ci
```
### Build
```
npm run build
```
### Run
Open in a browser: 
```
open ../docker/ui/index.html
```

### Build & run
```
npm run watch
```

## Backend

### Build
```
cd backend
npm ci
npm run build
```

### Start server
```
npm start
```
